# Workshop 05 - Building a REST API with Express and MongoDB
Tässä projektissa opetellaan rakentamaan REST API käyttäen Expressiä ja MongoDB:tä Mongoose-kirjaston avulla. API mahdollistaa blogipostausten luomisen, lukemisen, päivittämisen ja poistamisen.
## Tavoitteet
Harjoitella REST API:n rakentamista Expressillä ja MongoDB:llä blogipostausten CRUD-toiminnallisuuksia varten.
